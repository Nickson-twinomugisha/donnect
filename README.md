# Donnect — Blood Donation Hub

This repository contains the front-end for Donnect, a simple blood-donation hub built with Vite, React, and TypeScript.

Quick start

```bash
# install deps
npm install

# start dev server
npm run dev
```

Tech stack

- Vite
- React
- TypeScript
- Tailwind CSS

License & notes

This repo was scaffolded from a starter template and stripped of external tooling integrations.
